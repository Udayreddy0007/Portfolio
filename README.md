# Portfolio

ServiceNow Developer portfolio built with React, Vite, and Tailwind CSS.

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

## Deploy on Vercel

Connect this repository to Vercel. No root directory override is needed — `package.json` is at the repo root.

- **Framework:** Vite
- **Build command:** `npm run build`
- **Output directory:** `dist`
